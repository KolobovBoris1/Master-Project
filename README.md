Объектом исследования является система управления синхронного двигателя с постоянными магнитами.

Цель работы – проектирование системы в виде математической модели, максимально отражающую физическую модель для проведения экспериментов по настройке двигателя.

Методами исследования - анализ информационных источников, анализ устройства сервоконтроллера, анализ прикладных программ.

В результате исследования было выбрано приложение для моделирования системы на базе MATLAB. 
С помощью MATLAB была собрана математическая модель с рабочими контурами регулирования по аналогии с реальным сервоконтроллером.
